# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [0.1.0] - 2025-07-23
### Adicionado
- Inicial release do pacote Python **enderecobr**.
- Módulos de padronização:
  - `padronizar_bairros`
  - `padronizar_ceps`
  - `padronizar_complementos`
  - `padronizar_enderecos`
  - `padronizar_estados`
  - `padronizar_logradouros`
  - `padronizar_logradouros_completos`
  - `padronizar_municipios`
  - `padronizar_numeros`
  - `padronizar_tipos_de_logradouro`
- Módulos utilitários:
  - `codigos` (carregamento de códigos de estados e municípios)
  - `erro` (exceções padronizadas)
  - `warning` (emissão de avisos)
  - `mensagem` (logging de informação)
- Arquivos de dados:
  - `codigos_estados.json`
  - `codigos_municipios.csv`
